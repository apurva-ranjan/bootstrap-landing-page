Responsive Bootstrap Landing Page

Project Overview

This project is a fully responsive, single-page landing website built using HTML5, Bootstrap 5 (CDN), custom CSS, Vanilla JavaScript, and Font Awesome icons. It demonstrates professional frontend development practices including semantic structure, mobile-first responsive design, dynamic interactions, and deployment via GitHub Pages.

Live Demo
[https://apurva-ranjan.github.io/bootstrap-landing-page/]

Technologies Used

HTML5 – Semantic structure with <header>, <section>, <footer>.
Bootstrap 5 – Responsive grid, navbar, modal, alerts.
Custom CSS – Branding, color themes, spacing, dark/light mode, transitions.
Vanilla JavaScript – Form validation, dynamic alerts, modal control, scroll-to-top button, dark/light mode toggle, loading spinner.
Font Awesome – Icons for navbar, service cards, buttons, and footer social icons.
Git & GitHub – Version control and public repository hosting.
GitHub Pages – Live deployment of the website.

Features

1.Navigation / Header

Responsive Bootstrap navbar with collapsible menu for mobile devices.
Includes Font Awesome icons in navigation links.
Smooth scrolling to page sections.

2.Hero Section

Full-width banner with heading, subheading, and Call-to-Action (CTA) button.
CTA triggers a Bootstrap modal popup.

3.About Section

Two-column layout: image + descriptive text.
Responsive stacking on smaller screens.

4.Services Section

Minimum 2 service cards, each with Font Awesome icons.
Clean, professional layout and spacing.

5.Contact Form

Fields: Name, Email, Message.
JavaScript validation with dynamic Bootstrap alerts:
Success alert for valid submissions.
Danger/warning alert for validation errors.
Alerts are dismissible.

6.Footer

Copyright text.
Social media icons (Font Awesome).

7.Extra Features

Scroll-to-top button appears on scroll with smooth behavior.
Dark/Light mode toggle implemented via JavaScript and CSS.
Loading spinner visible on initial page load and removed after content is ready.

Installation & Setup

Clone the repository:

git clone https://github.com/apurva-ranjan/bootstrap-landing-page.git
Open index.html in a web browser.
Ensure internet connection to load Bootstrap 5 and Font Awesome via CDN.
Modify css/style.css and js/script.js to customize styles and interactions.
